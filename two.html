Data Types

  • Primitive Data Types (7 of them, think of them as categories)
    1. String
      • characters enclosed in single quotes '', double quotes "", or backticks ``
        • i.e. "" (empty string), "welcome", 'no one belongs here more than you', `good evening!`, `How was your day???`, "ðŸ˜ƒðŸ˜Ž", '777 lucky', etc.
      • .length property
      • Concatenation (combining)
      • Interpolation
        • Template Literals
    2. Number
      • Used for positive or negative integers and decimal values i.e. 0, 345, -23.4, 3.14, 42e-1 (scientific notation)
      • Different base representations:
        • binary: begin with 0b i.e. 0b110
        • hexadecimal: begin with 0x i.e. 0x11
        • octal: begin with 0o i.e. 0o11
      • Infinity
      • NaN (Not a Number)
    3. Boolean
      • true - on state, yes
      • false - off state, no
    4. Undefined
      • undefined no value assigned
    5. Null
      • null "absence of value"
    6. BigInt (not commonly used)
      • Used to represent very large integers
      • BigInt was added since Number types can't accurately do arithmetic with huge numbers
      • Larger than Number.MAX_SAFE_INTEGER and smaller than Number.MIN_SAFE_INTEGER
      • Denoted by suffixing a number with an "n" i.e. 9007199254740999n to the end of an integer
    7. Symbol (not commonly used)
      • Creates a unique value, no two symbols are ever the same
      • Symbol() or Symbol("favorite");
  • Objects
    We are able to put together primitive data types to compose larger data structures called objects. There many different type of objects such as Objects, Arrays, and Functions. More about these in future days.

Operators

  • Assignment
    • = assignment
    • += addition assignment
    • -= subtraction assignment
    • *= multiplication assignment
    • /= division assignment
  • Arithmetic
    • + addition
    • - subtraction
    • * multiplication
    • / division
    • % modulo (remainder division)

    Some examples:
    • typeof "Bitwise Industries";
